# archive-cn-cctv

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式保存，可离线浏览或部署到任意静态托管服务。

## 项目简介

该仓库旨在提供一个集中存放历史页面快照的空间，用于记录和回顾特定时期的网页内容。所有页面均为独立的 HTML 文件，不依赖外部数据库或后端服务。

## 目录说明

```
├── pages/         # 存放归档的 HTML 页面
├── assets/        # 页面引用的样式、脚本、图片等资源
└── index.html     # 可选的导航页面（如有）
```

- `pages/`：每个子目录或文件名代表一个归档条目，内部包含对应的 HTML 文件及其附属资源。
- `assets/`：公共资源目录，存放多个页面共用的文件。
- `index.html`：如果仓库包含首页，则用于列出归档页面索引。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开。
- 归档内容可能包含文本、图片、样式及脚本，但不会执行任何网络请求或外部调用。
- 页面保存时尽可能保留原始样式与布局，但因技术限制，部分交互功能可能无法完全还原。
- 归档时间点以页面保存日期为准，具体日期可在提交记录或文件命名中查看。

## 维护说明

- 归档内容不定期更新，每次提交对应一批新增或更新的页面。
- 仓库不接受主动提交的归档请求，仅由维护者按需添加。
- 如发现页面显示异常或链接失效，欢迎提交 Issue 说明情况。
- 本仓库仅用于存档目的，不反映维护者对归档内容的立场或观点。

## 使用方式

你可以通过以下方式使用本仓库的内容：

1. **直接浏览**：在浏览器中打开 `pages/` 目录下的 HTML 文件即可查看。
2. **本地部署**：将整个仓库克隆到本地，使用任意 HTTP 服务器（如 Python 的 `http.server`）提供服务。
3. **托管部署**：将仓库内容部署到 GitHub Pages、Netlify、Vercel 等静态托管平台。
